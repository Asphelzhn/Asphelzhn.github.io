1.Arrays类

1.1 概述

java.util.Arrays 此类包含用来操作数组的各种方法，比如排序和搜索等。其所有方法均为静态方法，调用起来
非常简单。

1.2 操作数组的方法

  1. public static String toString(int[] a) ：返回指定数组内容的字符串表示形式。

  2. public static void sort(int[] a) ：对指定的 int 型数组按数字升序进行排序。
