Tomcat启动乱码问题

发表于 2019-04-15 | 分类于 Java |

Tomcat启动乱码

使用的是tomcat8.5，环境变量配置没问题，可以正常使用start.bat启动，但是启动后的命令行日志显示乱码。将tomcat配置到idea中也有同样的问题。可以通过修改Tomcat配置文件解决。

找到tomcat/conf/目录下的logging.properties

找到 java.util.logging.ConsoleHandler.encoding = utf-8

更改为 java.util.logging.ConsoleHandler.encoding = GBK

保存，重启Tomcat即可
